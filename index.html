Подводный робот для поиска и мониторинга загрязнений

Описание продукта

Подводный робот от компании 'Общество с ограниченной ответственностью «Андербот-Кубань»', созданный для эффективного поиска загрязнений в водоемах, имеет следующие характеристики:

  • Максимальное количество замеров за одно погружение: 1
  • Трансляция видео: В режиме реального времени
  • Максимальное время работы: 2 часа
  • Способ связи: проводной
  • Питание: Li-Po батарея 12В
  • Максимальная глубина: 50 метров

Управление роботом

Управление роботом осуществляется с помощью пульта. Описание пульта дистанционного управления:

Пульт имеет проводной способ связи. Провод подключается через разъем к пульту и к роботу

В верхней части расположен дисплей для вывода изображения в режиме реального времени

В нижней части находится дисплей для вывода технический информации

Основными органами управления являются джойстики

Питание пульта происходит от аккумулятора li-ion

Зарядка пульта происходит через разъем type c
